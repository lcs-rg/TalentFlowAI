# Vision

## Produto

**TalentFlow AI** (nome provisório) é uma plataforma SaaS moderna de recrutamento inteligente.

Conecta empresas e candidatos através de recursos de Inteligência Artificial.

## Filosofia

A IA nunca substitui completamente o usuário. Ela atua como um **copiloto**.

## Prioridades

1. Simplicidade
2. Velocidade
3. Experiência premium
4. Escalabilidade
5. Arquitetura limpa
6. IA aplicada para resolver problemas reais

## Portais

### Empresa

- Gerenciamento de vagas
- Pipeline
- Candidatos
- Entrevistas
- Dashboard
- Automações

### Candidato

- Currículo
- Melhoria com IA
- Compatibilidade com vagas
- Aplicações
- Acompanhamento de processos

## Princípio Imutável

Toda decisão técnica futura deve respeitar esta visão.
