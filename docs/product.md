# Product

## Definição

TalentFlow AI é uma plataforma SaaS de recrutamento inteligente.

## Modelo de Negócio

**B2B**. Empresas contratam uma assinatura para utilizar o ATS.

Candidatos utilizam gratuitamente o portal profissional.

## Objetivo

Conectar empresas e profissionais utilizando Inteligência Artificial para aumentar a eficiência do processo seletivo.

## Portais

### Empresa

- Gestão de vagas
- Gestão de candidatos
- Pipeline
- Entrevistas
- Dashboard
- IA

### Candidato

- Perfil profissional
- Currículo
- Aplicações
- Compatibilidade com vagas
- Melhorias de currículo utilizando IA

## Posicionamento

A IA é uma funcionalidade do produto e nunca o produto em si.

Toda nova funcionalidade deve reforçar este posicionamento.
